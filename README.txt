README

This repository is created for our Data Analysis project in University of Applied Science. 
This repository contains the python files used in order to transfer the data from my local Visual Studios Code to MongoDB in order to create graphs in MongoDB, a Spark file that contains graphs of its own, as well as the csv file of the data itself. 
The goal of this project is to analyse data about patients and create graphs correlating to each other in order to find similarities and interesting trends.
